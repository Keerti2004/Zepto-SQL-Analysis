## Project Overview
This project analyzes Zepto product data using SQL to uncover pricing trends, discounts, inventory insights, and category performance.

## Tools Used
- SQL
- PostgreSQL / SQL Server
- Data Analysis

## Dataset Columns
- SKU ID
- Product Name
- Category
- MRP
- Discount Percentage
- Available Quantity
- Discounted Selling Price
- Weight
- Out of Stock Status

## Analysis Performed
- Data Cleaning
- Null Value Checks
- Category-wise Analysis
- Discount Analysis
- Inventory Analysis
- Product Pricing Insights

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE WHEN
- CTEs
- Joins
- Window Functions

## Key Insights
- Identified categories with highest discounts
- Analyzed stock availability across products
- Compared MRP and selling prices
- Explored category-wise product distribution
